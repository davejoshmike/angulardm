    Note:

    By using the static keyword
    ex:

    var express = require('express');
    var app = express();
    app.use(express.static('public'));

    we are defining the area in the filesystem that serves up imgs, javascript (client side), html, css, etc., but no server-side code. Essentially, the 'public' folder holds only things that will potentially be served up to the client.
